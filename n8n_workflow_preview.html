My n8n Workflow Preview

Explanation:

To embed an n8n workflow preview on your website, you can use the <n8n-demo> web component. This component simplifies the process by allowing you to directly embed the workflow JSON within your HTML.

Steps to implement:

  1. Include the necessary scripts: Add the following script tags in the <head> section of your HTML file. These scripts load the web components polyfill and the n8n-demo component itself. 
    <script src="https://cdn.jsdelivr.net/npm/@webcomponents/webcomponentsjs@2.0.0/webcomponents-loader.js"></script>
<script src="https://www.unpkg.com/lit@2.0.0-rc.2/polyfill-support.js"></script>
<script type="module" src="https://cdn.jsdelivr.net/npm/@n8n_io/n8n-demo-component/n8n-demo.bundled.js"></script>

  3. Customize with attributes: The <n8n-demo> component offers several attributes to customize its behavior and appearance:
    • workflow: (Required) The JSON string of your n8n workflow.
    • frame: Set to "true" to display a frame around the canvas with a code and copy button.
    • clicktointeract: Set to "true" to add a button that users must click before interacting with the canvas, which can improve scrolling on pages with multiple demos.
    • theme: Specifies the theme for the n8n workflow. Accepts "light" or "dark".
    • tidyup: Set to "true" to automatically arrange nodes in a clean, organized layout when loaded.
    • disableinteractivity: Set to "true" to prevent user interaction with the workflow entirely.
    • hidecanvaserrors: Set to "true" to hide node errors on the canvas.
  4. Styling: You can style the <n8n-demo> component using CSS to control its width, height, borders, and other visual properties, as demonstrated in the example HTML.
